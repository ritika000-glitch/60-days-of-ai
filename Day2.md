# Day 2 – Prompt Engineering

## Objective
Compare the outputs generated by a Lazy Prompt and an Engineered Prompt.

## Lazy Prompt
Create an image explaining Prompt Engineering.

### Observation
The output was simple and lacked details.

---

## Engineered Prompt
Create an educational infographic explaining Prompt Engineering.

### Observation
The output was more detailed, structured, and visually appealing.

---

## Comparison

| Lazy Prompt | Engineered Prompt |
|------------|------------------|
| Basic output | Detailed output |
| Less information | More information |
| No structure | Well organized |
| Generic design | Better visuals |
| Limited explanation | Clear explanation |

---

## Key Learnings

- Better prompts produce better results.
- Context improves AI responses.
- Clear instructions increase accuracy.
- Prompt engineering helps generate high-quality outputs.
